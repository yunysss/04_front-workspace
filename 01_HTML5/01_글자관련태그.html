문단을 나누는 태그 : p, pre

문단 영역을 나누는 태그로는 p태그와 pre태그가 있다. p태그는 줄바꿈하고자 한다면 별도의 태그를 작성해야됨
그리고 공백은 한 개의 공백만을 표시하기 때문에 별도의       기호문구를 기술해야됨 

            pre태그는 시작태그에서부터 종료태그까지
            입력된 내용 그대로를 표현하는 태그
            줄바꿈이라던가        여러개의 공백을 바로 표현함

strong : 글자를 굵게 표시하는 태그 + 스크린리더

u : 글자에 밑줄이 그어지는 태그

s : 글자에 취소선을 넣어주는 태그

em : 글자를 기울여서 표시하는 태그

i : 글자를 기울여서 표시하는 태그

mark : 형광펜 효과를 주는 태그

small : 글자를 작게 표현해주는 태그

기본글자에 sub : 아래첨자를 나타내는 태그와 sup : 윗첨자를 나타내는 태그
