폼 관련 태그

form태그내에 submit버튼 클릭시 해당 form안에 작성된
사용자가 입력한 값들을 서버로 넘기면서 요청하는 역할 수행

<form action="" method="">

* action 속성 : 해당 form에 사용자가 입력한 값들을 전달하면서 요청할 서버의 경로 제시
* method 속성 : 요청 전송 방식을 지정하는 속성 (get(기본값)/post)
> get방식 : 사용자가 입력한 값들이 요청시 url에 노출되는 방식 / 전달값에 대한 크기 제한 있음
> post방식 : 사용자가 입력한 값들이 요청시 url에 노출되지 않는 방식 / 크기 제한 없음

* 라디오버튼과 체크박스 관련 input태그의 타입

라디오버튼(type="radio")

- 제시한 여러개의 값들 중에서 오로지 한 개만 선택해야할 때
(name속성값이 똑같은 것들끼리 하나의 그룹으로 지어짐 => 그룹내에 한가지만 선택 가능)
- 제출시 현재 선택된(checked) 라디오버튼의 value값이 넘어감
(즉, 해당 라디오버튼 선택시 어떤 값을 넘길 것인지 value값으로 명시)

성별 :

체크박스(type="checkbox")

- 제시한 여러개의 값들 중에서 여러개 선택가능할 경우
- 제출시 현재 checked된 체크박스의 key+value세트가 넘어감

* select태그와 option태그

- 사용자에게 직접 키보드로 값을 입력받는게 아니라 내가 제시한 여러개의 옵션들(목록) 중에서 선택할 수 있게끔 제공
- 제출시 현재 선택된(selected) option 값이 넘어감
각 option에 value 값을 명시하지 않았을 경우 content영역값이 넘어감
value 값을 명시했을 경우 value값이 넘어감

HTML5에서 새로 추가된 기능 datalist

- 사용자에게 직접 키보드로 값을 입력받을 수도 있고 목록에서 선택할 수도 있음
